# 机器学习

## 一、 什么是机器学习

定义：赋予计算机在没有明确编程的情况下学习能力的研究领域

方式：输入数据、进行计算、调整参数

机器学习是实现人工智能的一种途径。

| 类型           | 核心特点                                            | 典型应用                         |
| -------------- | --------------------------------------------------- | -------------------------------- |
| **监督学习**   | 训练数据带标签（输入 + 预期输出）                   | 图像分类、房价预测、垃圾邮件识别 |
| **无监督学习** | 训练数据无标签，自动挖掘数据规律                    | 客户分群、异常检测、推荐系统     |
| **强化学习**   | 模型通过与环境交互，以 “奖励 / 惩罚” 为目标优化策略 | 自动驾驶、AlphaGo、机器人控制    |
|**半监督学习**|训练数据一部分有标签，大部分无标签|标签数据获取成本高的情况（比如医学影像标注，需要专业医生）|



## 二、 深度学习


深度学习是机器学习的一个子集,也就是说深度学习是实现机器学习的一种方法。
深度学习是机器学习中一种基于对数据进行特征学习的算法。
深度学习是基于人工神经网络,深度是指网络中使用多层,每层都通过非线性变换处理数据,并逐渐提取出更复杂、更抽象的特征。

## 三、神经网络

定义：人工神经网络，也简称为神经网络(NN),是一种模仿生物神经网络结构和功能的计算模型。

#### 1.架构构成：

| 层级       | 作用                                                         | 示例（房价预测）                                             |
| ---------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **输入层** | 接收原始数据，仅传递数据，不做任何计算，神经元数量 = 输入特征的维度。 | 2 个神经元：分别对应 “面积”“卧室数”                          |
| **隐藏层** | 对输入数据做核心的特征提取和变换，是网络 “学习能力” 的核心，可多层。 | 比如设 4 个神经元（可自定义），负责学习 “面积→价格”“卧室数→价格” 的复杂关系 |
| **输出层** | 输出最终结果，神经元数量 = 预测目标的维度。                  | 1 个神经元：输出预测的房价（万元）                           |

#### 2. 基本组成：人工神经元

人工神经元是神经网络的最小计算单元，它接收多个输入，对每个输入分配不同的权重，进行加权求和后，通过一个**激活函数**产生输出。

- **输入**：来自上一层神经元的输出或原始数据。
- **权重**：代表输入的重要程度，在训练过程中不断调整。
- **偏置**：一个额外的可学习参数，用于调整激活函数的触发阈值。
- **激活函数**：引入非线性，让网络能够学习复杂的模式，常见的有 ReLU、Sigmoid、Tanh 等。

#### 3.数据集的划分：

| 数据集 | 核心作用                                                     |
| ------ | ------------------------------------------------------------ |
| 训练集 | 用来训练模型，调整模型的权重参数，让模型学习数据中的规律。   |
| 验证集 | 用来评估模型在训练过程中的效果，调整超参数（比如隐藏层神经元数量、学习率），选择最优模型。 |
| 测试集 | 用来最终评估模型的泛化能力，测试集的数据从未被模型见过，结果是模型的最终 “成绩单”。 |

#### 4.网络的推理：前向传播

前向传播是神经网络进行预测的过程，数据从输入层流向输出层，逐层计算，最终得到预测结果。

①神经网络的前向传播

1. 输入层将原始数据传递给第一个隐藏层。
2. 隐藏层的每个神经元对输入进行加权求和，并通过==激活函数==产生输出，传递给下一层。
3. 这个过程在所有隐藏层中重复进行。
4. 最后，输出层神经元计算出最终的预测值（如分类概率、回归值）。

总结：前向传播就是 “用当前的权重和偏置，对新数据做一次完整的计算，得到预测结果”。

激活函数：是神经元的**核心开关**，作用是给线性计算的结果添加**非线性特性**

| 激活函数    | 公式              | 通俗理解                                            | 适用场景                                           |
| ----------- | ----------------- | --------------------------------------------------- | -------------------------------------------------- |
| **Sigmoid** | f(x)=1+e−x1       | 把任意输入压缩到 `(0,1)` 之间，像一个 “概率开关”    | 二分类问题的输出层、早期神经网络                   |
| **Tanh**    | f(x)=ex+e−xex−e−x | 把输入压缩到 `(-1,1)` 之间，输出以 0 为中心         | 隐藏层（比 Sigmoid 收敛更快）                      |
| **ReLU**    | f(x)=max(0,x)     | 输入为正就 “放行”，为负就 “关闭”，像一个 “单向阀门” | 目前最常用的隐藏层激活函数（计算快、缓解梯度消失） |
| **Softmax** | f(xi)=∑j=1nexjexi | 把多个输出值转换成**概率分布**（所有输出和为 1）    | 多分类问题的输出层                                 |

一个神经元是神经网络的最小单元，可以类比成一个带开关的计算器。

#### 5.网络的训练：反向传播

反向传播是神经网络 “学习” 的核心机制，用于不断调整权重和偏置，使预测结果尽可能接近真实值。

1. **计算损失**：首先通过前向传播得到预测值，然后用损失函数（如均方误差、交叉熵）计算预测值与真实值之间的误差。
2. **反向传播误差**：从输出层开始，将误差逐层反向传递到前面的每一层，计算每个权重和偏置对总误差的 “贡献”（即梯度）。
3. **更新参数**：使用优化器（如梯度下降、Adam）根据梯度更新权重和偏置，使总误差逐步减小。
4. **迭代训练**：重复前向传播、计算损失、反向传播、参数更新的过程，直到损失收敛或达到预设的训练轮数。

通过上述过程，神经网络不断 “修正” 自己的判断，最终学会从数据中提取规律。

#### 6.损失函数：判断是否是好的回归分析

损失函数（Loss Function）的作用是**量化预测值 y^ 和真实值 y 的差距**，输出一个非负的 “损失值”：

- 损失值越小 → 预测越准
- 损失值越大 → 预测越差

## 四、过拟合问题

#### 1、过拟合

定义：指模型在**训练数据上表现极好**（损失值极低、预测准确率接近 100%），但在**从未见过的测试 / 新数据上表现极差**的现象。

原因：模型复杂度 > 数据信息量

#### 2、过拟合的解决方法——正则化

定义：在神经网络训练过程中，**通过在损失函数中加入 “惩罚项”**，限制模型参数（主要是权重 w）的大小，从而降低模型复杂度、避免过拟合的技术。

#### 3.数据归一化

用途：神经网络的参数更新依赖**梯度下降**，而不同特征的数值尺度差异会导致两个关键问题：

1. **特征 “不公平”**：数值大的特征（比如房价中的 “面积”：50-200）会主导梯度计算，数值小的特征（比如 “房间数”：1-5）被忽略，模型学不到真实规律。
2. **收敛速度慢**：梯度下降在尺度不均的特征空间中会走 “之字形” 路线，需要更多迭代才能收敛到最优解。

方法：

| 方法           | 输出范围          | 对异常值的敏感度           | 适用场景                     |
| -------------- | ----------------- | -------------------------- | ---------------------------- |
| Min-Max 归一化 | [0,1]（可自定义） | 高（异常值会改变 min/max） | 特征范围明确（如图像、像素） |
| Z-Score 标准化 | 无固定范围        | 低（仅影响均值 / 标准差）  | 大多数神经网络训练场景       |

作用：

**加速模型收敛**：梯度下降在尺度统一的特征空间中沿直线收敛，迭代次数减少 50% 以上。

**提升模型精度**：让所有特征公平参与学习，避免数值大的特征主导参数更新。

**避免数值溢出**：将大数值特征缩放到小范围，防止权重计算时超出浮点数范围。

**适配激活函数**：比如 Sigmoid 函数在 | x|>5 时梯度趋近于 0，归一化后特征值落在 [-2,2] 区间，激活函数能有效发挥作用。

#### 4.数据清洗

定义：数据清洗是**预处理原始数据，去除噪音和错误**的过程。

目标：

1. 修复数据中的**缺失、错误、重复**问题
2. 剔除**异常值**和**无关数据**
3. 统一数据**格式和标准**
4. 让最终的数据满足模型输入要求

#### 5.Batch Size 

定义：指**每次迭代时，模型一次性读取并用于计算梯度的样本数量**。

类型：

| Batch Size 类型       | 取值范围     | 优点                                                         | 缺点                                                         | 适用场景                             |
| --------------------- | ------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------ |
| 小批次（Small Batch） | 8/16/32      | 1. 梯度更新频繁，模型收敛更快；2. 梯度有 “噪声”，易跳出局部最优，泛化能力更强；3. 内存占用小 | 1. 训练时间长（迭代次数多）；2. 梯度波动大，损失曲线震荡     | 小数据集、深度学习（CNN/RNN）        |
| 大批次（Large Batch） | 64/128/256   | 1. 梯度更稳定，损失曲线平滑；2. 可利用 GPU 并行计算，单轮迭代速度快；3. 减少迭代次数 | 1. 易陷入局部最优，泛化能力差；2. 内存占用大；3. 收敛慢      | 大数据集、工程化部署（追求训练效率） |
| 全批次（Full Batch）  | 等于样本总数 | 1. 梯度是全局最优的；2. 损失曲线最平滑                       | 1. 内存占用极大（仅适合小数据集）；2. 极易过拟合；3. 收敛极慢 | 仅理论场景（比如线性回归）           |

定义：学习率是梯度下降中**参数更新的 “步长”**，决定每次迭代时权重 / 偏置的调整幅度。

## 五、梯度下降算法

定义：通过一次次的逼近，找到最小的损失函数和最优的参数

为什么需要梯度下降?
因为有些函数可以直接根据公式求出其最小值点,但是有些函数求解的难度太大。所幸,他的图像是一座山谷,所以我们可以通过梯度下降法一步步滚动到谷底(数学:逼近最小值点)。

梯度向量:指向函数值增长最快的方向
所以如果取负的梯度向量,则是指向函数之减少最快的方向
