# 矩阵

## 一、矩阵

​		矩阵是由 m×n 个数排列成的**m 行 n 列的矩形数组**，记为 A~m×n~。矩阵运算的核心是**维度匹配规则**，所有运算都需要满足维度要求才能进行。

随机矩阵：它的每一项都是一个表示概率的非负实数

​		右随机矩阵是实方阵，其中每一行求和为1

​		左随机矩阵是实方阵，其中每一列求和为1

​		双随机矩阵是非负实数方阵，每个行和列求和均为1

对角矩阵:一个除了主对角线之外的元素皆为0的矩阵。对角线上的元素可以为0或其他值。

数量矩阵：对角线上元素相等的对角矩阵称为数量矩阵

单位矩阵：对角线上元素全为1的对角矩阵称为单位矩阵

## 二、运算

加和 数乘：

| 运算         | 定义                                              |
| ------------ | ------------------------------------------------- | -------------------------------------- |
| **矩阵加法** | 同维度矩阵对应位置元素相加，A+B 要求 A,B 均为 m×n |
| **数乘**     | 常数 k 乘以矩阵每个元素，kA 无维度限制            |

乘法：

**左矩阵的列数 = 右矩阵的行数**，否则无法相乘。

**性质**：

- 不满足交换律：AB\\=BA（即使都能相乘）
- 满足结合律：(AB)C=A(BC)
- 满足分配律：A(B+C)=AB+AC

**转置**：

A^T^ 是将 A 的行和列互换，(A^T^)~ij~=A~ji~，性质 (AB)^T^=B^T^A^T^

矩阵与向量的关系：向量是特殊的矩阵，1行n列和n行1列的分别为行向量和列向量

## 三、逆矩阵

定义：设A是n阶**方阵**,若存在n阶方阵B,使AB=BA=E,则称A是可逆矩阵,B为A的逆矩阵,记作A^-1^,即A^-1^=B。

性质：

**前提**：仅方阵且行列式≠0（可逆矩阵）才有逆，逆唯一；

**定义恒等式**：A⋅A^−1^=A^−1^⋅A=I（I为单位矩阵）；

**自反性**：(A−1)−1=A（逆的逆是原矩阵）；

**乘积逆**：(A⋅B)^−1^=B^−1^⋅A^−1^（乘积的逆 = 逆的反向乘积）；

**数乘逆**：(k⋅A)^−1^=k^-1^⋅A^−1^（k\\=0，数乘的逆 = 倒数乘原逆）；

**转置逆**：(A^T^)^−1^=(A^−1^)^T^（转置的逆 = 逆的转置）。



## 四、矩阵求导

| 求导类型   | 结果维度                   | 核心示例（x为向量，A为常数矩阵） |
| ---------- | -------------------------- | -------------------------------- |
| 标量对向量 | 与向量同维度               | ∂x∂(aTx)=a                       |
| 向量对标量 | 与向量同维度               | ∂k∂(kx)=x                        |
| 向量对向量 | 行 × 列（分子行 × 分母列） | ∂x∂(Ax)=AT                       |
| 矩阵对矩阵 | 按元素求导（逐元素）       | ∂B∂(AB)=AT（简化版）             |

## 一、点乘（内积）

对 n维向量 ，a=(a1,a2,...,an),b=(b1,b2,...,bn)，点乘结果为标量：

**a**⋅**b**=a^1^b^1^+a^2^b^2^+...+a^n^b^n^=∣a∣∣b∣cosθ

θ是两向量的夹角。

## 三、线性变换

设 V,W 是数域 F 上的线性空间，若映射 T:V→W 满足：

1. **可加性**：T(u+v)=T(u)+T(v)

2. 齐次性：T(ku)=kT(u)（k为任意标量）

   则称 T 是线性变换。
